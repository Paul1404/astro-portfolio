Repositories einzeln in der GitHub-Oberfläche zu archivieren ist mühsam, und einen wirklich vertrauenswürdigen Rückgängig-Button gibt es nicht. Dieses Tool verpackt das GitHub CLI in ein sicheres, interaktives Batch-Werkzeug.

### Was es macht

- Ermöglicht die interaktive Auswahl von Repositories, die dann gesammelt archiviert oder de-archiviert werden.
- Unterstützt einen Dry-Run, mit dem man genau sehen kann, was sich ändern wird, bevor etwas passiert.
- Bestätigt destruktive Aktionen und protokolliert, was ausgeführt wurde.

### Warum es wichtig ist

Einen vollen GitHub-Account aufzuräumen wird zu einem Zwei-Minuten-Job statt eines nachmittäglichen Klickmarathons, ohne die Angst, das falsche Repository zu erwischen.
