Mailcow zu verwalten bedeutet normalerweise die Admin-UI oder rohe API-Aufrufe. Dieser Skill bringt Claude Code bei, das für einen zu übernehmen, sodass man einen Mailserver in normaler Sprache steuern kann.

### Was es macht

- Verpackt gängige Mailcow-Operationen als Claude Code Skill, der die Mailcow API aufruft.
- Ermöglicht es, Mailboxen, Domains und Aliase im Gespräch zu erstellen und zu verwalten.
- Hält die API-Details hinter einer sauberen Schnittstelle verborgen, statt sich Endpoints merken zu müssen.

### Warum es wichtig ist

Ein praktisches Beispiel dafür, wie man einen KI-Agenten mit echter Infrastruktur-Tooling verbindet, wo die Aktionen echte Konsequenzen haben.
