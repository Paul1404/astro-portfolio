E-Mail-Clients sind die schlimmste Rendering-Zielplattform überhaupt. Outlook verwendet die Word-Engine, Gmail streift die Hälfte des CSS heraus, und Apple Mail invertiert Farben im Dark Mode einfach ohne Vorwarnung. ESG erzeugt Signaturen, die das alles überstehen.

### Was es macht

- Generiert pixelgenaue Signaturen für Outlook (Word-Renderer), Gmail im Web und mobil, Apple Mail inklusive automatischer Dark-Mode-Invertierung, Outlook 365 / OWA und Yahoo.
- Produziert sauberes, tabellenbasiertes HTML, das sich ohne Probleme direkt in einen Client einfügen lässt.
- Ermöglicht eine Vorschau des Ergebnisses, bevor es an eine ganze Organisation ausgerollt wird.

### Warum es wichtig ist

Eine einheitliche Signatur unternehmensweit einzuführen bedeutet normalerweise Copy-and-Paste-Chaos und ein unterschiedlich kaputtes Layout in jedem Postfach. ESG macht daraus einen wiederholbaren, vorhersehbaren Build-Schritt.
