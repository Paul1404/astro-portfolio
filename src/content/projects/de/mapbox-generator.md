Manchmal braucht man einfach ein sauberes Bild einer Route und kein interaktives Karten-Widget. Dieses Tool erzeugt druck- und webfertige statische Karten aus einer Routendefinition.

### Was es macht

- Rendert Routenkarten über die Mapbox Static Images API.
- Gibt Bilder in den richtigen Größen für Banner, Flyer und Webnutzung aus.
- Hält das Styling konsistent, sodass ein Satz von Karten wie aus einem Guss wirkt.

### Warum es wichtig ist

Ein kleines, fokussiertes Tool, das das manuelle Screenshot-und-Zuschneiden-Spiel beendet, sobald man eine Karte für ein Design braucht.
