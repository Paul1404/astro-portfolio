Sometimes you just need a clean picture of a route, not an interactive map widget. This produces print and web ready static maps from a route definition.

### What it does

- Renders route maps using the Mapbox Static Images API.
- Outputs images sized for banners, flyers and web use.
- Keeps the styling consistent so a set of maps looks like it belongs together.

### Why it matters

A small, focused tool that removes the manual screenshot-and-crop dance whenever you need a map for a design.
