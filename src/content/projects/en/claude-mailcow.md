Managing Mailcow normally means the admin UI or raw API calls. This teaches Claude Code how to do it for you, so you can manage a mail server in plain language.

### What it does

- Packages common Mailcow operations as a Claude Code skill that calls the Mailcow API.
- Lets you create and manage mailboxes, domains and aliases conversationally.
- Keeps the API details behind a clean interface instead of memorising endpoints.

### Why it matters

A practical look at wiring an AI agent into real infrastructure tooling, where the actions have real consequences.
