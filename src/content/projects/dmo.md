If you run a lot of repositories you end up drowning in Dependabot pull requests. Going through them one repo at a time is slow and you start ignoring them, which is exactly how you fall behind on security updates.

### What it does

- Authenticates with GitHub and pulls every open Dependabot PR across all your repositories.
- Shows them on a single dashboard so you can see your whole update backlog at once.
- Lets you approve and merge in bulk instead of clicking through each repo.

### Why it matters

Keeping dependencies current stops being a chore you avoid. One screen, a few clicks, and the backlog is gone.
