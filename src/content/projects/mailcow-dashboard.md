Running your own mail server is fine until something is slow or queues are backing up and you have no idea why. This gives Mailcow a proper monitoring view.

### What it does

- A ready to import Grafana dashboard covering the key Mailcow and mail server metrics.
- A clean dark theme layout inspired by VMware VCF, so it is easy to read at a glance.
- Drop-in monitoring instead of building panels from scratch.

### Why it matters

Visibility into a self-hosted mail server, without spending a weekend wiring up dashboards yourself.
